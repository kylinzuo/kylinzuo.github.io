好记性不如烂笔头，学习记录Markdown语法，方便后续查阅。[Markdown中文文档](https://markdown-zh.readthedocs.io/en/latest/)

### 标题
```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

### 字体
```
**这是加粗的文字**
*这是倾斜的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```
**这是加粗的文字**
*这是倾斜的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~

### 引用
在引用的文字前加>即可，引用也可以嵌套，如加两个>>、加三个>>>、加n个>>>
```
>这是引用的内容
>>这是引用的内容
```
>这是引用的内容
>>这是引用的内容
